Google Calendarデモアプリケーション

概要

GData API(Java)で開発中のGoogle Calendar用を読み書き(CRUD)するライブラリを
デバッグする為に作ったWebアプリケーションです。

今お持ちのGoogle AccountやGoogle AppsのAccountでログインして
カレンダーイベントを日付指定で検索する事が出来ます。
読み込んだカレンダーに編集権限があれば編集する事も可能です。

デモサイトはAuthSub認証を使用してますので
利用される方のアカウントおよびパスワードをアプリケーションから取得する事はありません。

技術的な要素

  • Google App Engine for Java
  • GData API
  • javascript
  • jQuery
  • Sigmagrid

動作環境

Google ChromeおよびFirefoxで動作の確認をしています。
IE8でも動作する事を確認していますがセキュリティの警告が出ます。

注意事項

  • 追加および更新機能は有効になっています。普段利用されているカレンダーでの利用は注意してください。
  • 安全を考えて削除処理はデプロイ時に無効にしています。
